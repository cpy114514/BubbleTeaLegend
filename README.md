# 🧋 Bubble Tea Legend

Bubble Tea Legend is a 2D survival action game where the drink you craft becomes your fighter.

Mix milk, red or green tea, and toppings like pearls to create your own build.  
Enter the battlefield and survive endless waves of enemies.  

Every ingredient changes how your tea fights — no two cups battle the same way.

---

## 🎮 Features

- 🧋 Craft-your-own battle build system
- ⚔ Auto-attack survival gameplay
- 🎯 Attack range and target detection
- 💥 Pearl projectile combat
- ❤️ Visual health system using liquid particles
- 📈 Endless difficulty scaling

---

## 🧠 Core Concept

Unlike traditional survival games, your upgrades are decided **before the battle**.

The recipe you design directly affects:
- Attack behavior
- Fire rate
- Build strategy
- Survivability

Craft first. Survive after.
